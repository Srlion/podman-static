# podman-static

## Install

```bash
curl -fsSL https://github.com/srlion/podman-static/raw/main/install.sh -o /tmp/install.sh && bash /tmp/install.sh
```

## Uninstall

```bash
curl -fsSL https://github.com/srlion/podman-static/raw/main/uninstall.sh -o /tmp/uninstall.sh && bash /tmp/uninstall.sh
```
